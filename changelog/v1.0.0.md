# Changelog - v1.0.0 (Initial Release)

All notable changes to **GitPilot** will be documented in this file.

## [1.0.0] - 2026-03-26

### Added
- **Core Architecture**: Modular TypeScript engine with `PluginRegistry` and `EventBus`.
- **AI Agents**: 
  - `TriageAgent`: Auto-labeling for issues.
  - `ReviewAgent`: AI-powered PR code reviews.
  - `ReleaseAgent`: Automated release note generation.
  - `DeduplicationAgent`: Semantic search for duplicate issues.
- **GitHub Integration**: Support for GitHub Apps, Webhooks, and REST/GraphQL API via Octokit.
- **Maintenance**: Auto-stale issue management and contributor welcome messages.
- **Deployment**: Docker and Docker Compose support for production environments.
- **Health Checks**: `/health` endpoint for monitoring.
- Implemented `GITHUB_WEBHOOK_SECRET` validation to prevent unauthorized event spoofing.
