# Security Policy

## Supported Versions

Currently, we only provide security updates for the latest stable version of GitPilot.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of GitPilot seriously. If you discover a security vulnerability, please do not open a public issue. Instead, send an email to security@gitpilot.io (or the maintainer's contact).

Please include:
- A description of the vulnerability.
- Steps to reproduce (Proof of Concept).
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
